# learnsite-deepseek

LearnSite 信息科技学习平台 DeepSeek AI 模块 Docker 化

## 功能特性

- AI 对话（流式输出）
- PPT 大纲生成
- 图片生成（pollinations 免费 / 智谱 AI）
- 文本转语音
- 图片文字识别（OCR）
- 中英翻译
- 文件上传

## 快速开始

```bash
# 1. 修改 docker-compose.yml 中的 API 配置
# 2. 启动服务
docker-compose up -d

# 3. 访问
http://localhost:2000
```
